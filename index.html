Claude Code Hook Stop

Claude Codeの「完了」「許可待ち」を音で知らせる

StopフックとPermissionRequestフックで、応答完了や権限要求のタイミングに通知音を鳴らす設定方法。ながら作業の取りこぼしを防ぐ。トースト通知や音声読み上げの応用も。

Claude Code Hook Cursor

Cursor内のClaude Codeチャットに日時を表示する

標準では出ない各メッセージの送信時刻を、UserPromptSubmitフックとグローバルCLAUDE.mdの組み合わせで返信冒頭に表示させる設定方法。汎用フックの応用例も。
